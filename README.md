# Dublin Shelf Finder (Extension)

This is the Chrome extension for the Dublin Shelf Finder. It adds buttons to Destiny Destiny, which users can click to open a map of their library that shows the correct location of the book.

Read more on the main [website](https://shelf-finder.com) or [repo](https://github.com/deryilz/shelf-finder).

Or check it out on the [Chrome Web Store](https://chromewebstore.google.com/detail/dublin-shelf-finder/lclnakicapgebmahgjdhgpgieaimnfmg).
